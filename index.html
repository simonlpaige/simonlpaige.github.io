ABOUT

Simon

Hi, I'm Simon.

I'm in Kansas City. I've been building websites for twelve years, and lately I've been building tools that help communities run themselves better. I answer my own phone. I don't sell your data. If you email me, I write back.

On the client side, every site I build is hand-coded and lives as plain files on a server. That means it's fast, it doesn't break when some platform pushes an update, and you keep the code forever even if you stop working with me. Local, boring, durable.

On the open-source side, I'm building Commonweave and Neighborhood OS because I think the software neighborhoods actually need doesn't exist yet, and I'm tired of waiting for someone else to make it.
